## <center>集合类型</center>

> 集合是一个不重复的集合

### 一、创建集合的方式
* 1、直接一个字符

  ```py
  set1 = set('hello word')
  ```
* 2、一个列表(元组,字典都可以)

  ```py
  set1 = set([1, 1, 3, 4])
  ```

### 二、集合的操作

* 1、`add()`添加

  ```py
  set1.add('hello')
  ```

* 2、`remove()`删除

  ```py
  set1.remove('hello')
  ```

* 3、集合的与, 用`&`

* 4、集合的或,用`|`

* 5、集合的与或,用`^`
